# Project Scope (Group 48)

## Project Goal
Build a lightweight TA recruitment system for BUPT International School using Servlet/JSP and text-file persistence.

## Technical Constraints
- Architecture: Java Servlet/JSP web application
- Runtime: Tomcat
- Storage: text files only (`.json`, `.csv`, `.txt`, `.xml`)
- Database: not allowed

## Iteration 1 Core Scope
- User login and register
- TA creates/updates profile
- TA views available jobs and applies
- MO posts jobs and reviews applicants
- TA checks application status

## Out of Scope for Iteration 1
- Advanced UI polishing
- Full AI integration
- Complex workload optimization

## Demo Path (Iteration 1)
1. Register and login as TA
2. Complete TA profile
3. Find and apply for a job
4. Login as MO and select/reject applicant
5. Login as TA and verify updated application status
